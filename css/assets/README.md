# Best Foods.sn

Site vitrine bilingue (Français/Anglais) pour un restaurant moderne à Dakar.  
Cuisine variée et savoureuse, boissons maison fraîches.  
Ce projet est conçu pour être simple, responsive et professionnel.

[![WhatsApp](https://img.shields.io/badge/WhatsApp-Reserve%20Now-25D366?logo=whatsapp&logoColor=white)](https://wa.me/221700077009)

---

## ✨ Fonctionnalités
- Page unique responsive (`index.html`)
- Menu de navigation clair (Présentation, Menu, Boissons, Localisation, Réservation)
- Galerie d’images (`assets/images/`)
- Carte Google Maps intégrée
- Formulaire de réservation + lien WhatsApp
- Design moderne avec CSS (`css/style.css`)
- Widget Google Translate (FR ↔ EN)
