# Best Foods.sn 🍽️

## 📌 Description
Site vitrine bilingue (Français/Anglais) pour le restaurant **Best Foods**.  
Ce projet présente le menu, la localisation et permet une réservation rapide en ligne.  
Il est conçu pour être responsive et accessible sur mobile et ordinateur.

## ✨ Fonctionnalités
- Page principale bilingue (`index.html`)
- Design moderne et responsive (`CSS/style.css`)
- Galerie d’images (`Actifs/images`)
- Bouton de réservation rapide (WhatsApp ou formulaire)
- Carte Google Maps intégrée
- Optimisation SEO avec balises meta
